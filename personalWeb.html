Field Projects

Project Title 1: ML Competition - Open-Problem: Single - Cell Perturbations

In the fast-paced world of single-cell analysis, the Kaggle competition titled "Open Problem: Single-Cell Perturbations" plays a vital role. This research paper outlines the goals of the competition and underscores its importance in enhancing our comprehension of how genes react to experimental changes at the single-cell level. A key method used in this competition is Differential Expression (DE) Modeling, which allows us to calculate alterations in gene expression when subjected to perturbations. It can be found here.

Project Title 2: GitHub Explorer

GitHub Explorer is an educational resource and guide for developers of all levels, aiming to demystify key concepts in GitHub. Whether you're a seasoned developer or just starting your coding journey, this project offers a clear and comprehensive explanation of GitHub repositories, branches, forks, clones, and the fundamental workflow.
The goal is to empower developers to confidently navigate GitHub, collaborate on open-source projects, and contribute to the world of software development. With easy-to-understand explanations and practical insights, GitHub Explorer is your go-to resource for mastering GitHub's core concepts. Explore, learn, and unlock the full potential of version control and collaboration in software development. The article can be found here.
